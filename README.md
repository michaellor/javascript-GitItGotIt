#Git It Got It
####Javascript Code Review for Epicodus, 03.18.2016

###by Michael Lor

##Description

This is an app used to search for user data when a github username is provided and submitted into a form. The app returns the name, email, number of repositories the user has, as well as a sampled list of repository names and descriptions.

This app was developed in javascript, with intended application of npm, gulp, and bower technologies.

##Setup

1) Clone repository to local drive.

2) Install NPM by running the following command through the terminal:

    npm install

3) Next install Bower from the terminal:

    bower install

4) Run Gulp Build to compile the scripts:

    gulp build

5) Run Gulp Serve to run the localhost server to run the page:

    gulp serve

6) Once the app is up and running, simply enter in github username into the form and submit.
